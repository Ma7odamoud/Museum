# Music Files

Place your 3 music files here. Supported formats: **MP3**, **WAV**, **OGG**

## Recommended File Names:
- `song1.mp3`
- `song2.mp3`
- `song3.mp3`

## Instructions:
1. Copy your music files into this folder
2. Update the file names in `/lib/audio-config.ts` to match your actual file names
3. The audio player will loop through all songs continuously

## Example:
```
public/
  music/
    ├── song1.mp3
    ├── song2.mp3
    └── song3.mp3
```

The songs will play in the order specified in the configuration file.
