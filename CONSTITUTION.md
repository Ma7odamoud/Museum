# Virtual Memory Museum - Constitution

## Project Overview
I am building a "Virtual Memory Museum" for me and my girlfriend.

## Tech Stack
- **Framework**: Next.js (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Database**: Prisma + PostgreSQL
- **Animations**: Framer Motion for high-end, smooth transitions

## Core Rules

### 1. Privacy & Access
- **Private site only**
- Access via password **"220810"** only
- **No login/signup system**

### 2. Architecture
- **"Rooms" concept**: Every memory is a room you "enter"
- Navigate through memories as if walking through museum rooms

### 3. Audio Experience
- **Global audio player**: 3 songs on continuous loop
- Audio **never stops** when changing pages
- Persistent playback across all navigation

### 4. Design Philosophy
- **Mobile-first design**
- Smooth, high-end animations using Framer Motion
- Premium, museum-quality aesthetic

---

**Every line of code written for this project must adhere to these principles.**
