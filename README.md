# Memory Rooms - Virtual Memory Museum

A private, password-protected virtual museum for cherished memories.

## Tech Stack

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Prisma** + PostgreSQL
- **Framer Motion**

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Set up your database:
```bash
npm run db:push
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000)
